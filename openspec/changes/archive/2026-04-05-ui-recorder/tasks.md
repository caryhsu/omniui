## 3. Python — Selector Inference

- [ ] 3.1 Create `omniui/recorder/` package (`__init__.py`)
- [ ] 3.2 Create `omniui/recorder/selector_inference.py`: implement `infer_selector(event: RecordedEvent) -> dict`
  - `fxId` non-empty → `{"id": fx_id}`
  - else text ≤ 40 chars → `{"text": text}`
  - else → `{"type": node_type, "index": node_index, "_fragile": True}`

## 4. Python — Script Generator

- [ ] 4.1 Create `omniui/recorder/script_gen.py`: implement `generate_script(events: list[RecordedEvent]) -> str`
  - Header: `# Generated by OmniUI Recorder\nfrom omniui import OmniUI\n\nclient = OmniUI.connect()\n`
  - `click` event → `client.click(<selector>)`
  - `type` event → `client.type(<selector>, text="<text>")`
  - Fragile selector → append `  # WARN: fragile selector`
